# Marmitex

Projeto web do Marmitex com frontend em HTML/CSS/JS e backend em Node.js com Express.

## Como rodar

No terminal, dentro da pasta do projeto:

```powershell
npm install
npm start
```

Depois abra no navegador:

```text
http://localhost:3000
```

## Como abrir no celular

1. Inicie o servidor no computador:

```powershell
npm start
```

2. Descubra o IP local do computador:

```powershell
ipconfig
```

3. Procure o `Endereço IPv4` da rede Wi-Fi.

4. No celular, conectado na mesma rede, abra:

```text
http://SEU-IP:3000
```

Exemplo:

```text
http://192.168.0.15:3000
```

## Observacoes

- O backend precisa estar rodando para cadastro e login funcionarem.
- O projeto usa as configuracoes do arquivo `.env`.
- Se for a primeira vez no projeto, rode `npm install` antes de `npm start`.
- O celular e o computador precisam estar na mesma rede Wi-Fi.
